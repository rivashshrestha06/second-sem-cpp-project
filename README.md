**Simple Banking System (C++ Project)**

This is a simple C++ program that simulates basic banking operations for a **savings account**. It allows you to create an account, deposit and withdraw money, and check the balance ‚Äî all through a command-line interface.

---

**Features**

- Create a new account with name, address, and auto-generated account number
- Deposit and withdraw money with validation
- View account details
- Uses **classes**, **constructors**, and **static data members**

---

**How to Compile & Run**

**On Windows (using g++):**
```bash
g++ main.cpp -o banking.exe
./banking
```
